# Substack Labo

Substackを中心に、ノート、記事、投稿候補、フォロー/フォロワー、アイデア、仮メモをまとめて管理する制作ワークスペースです。

## ページ

- トップ: 全体のまとめ
- ノートページ
- 記事ページ
- Substack投稿ページ
- フォローページ
- フォロワーページ
- アイデア保管ページ
- 仮メモページ

## できること

- ノートは本文、ステータス、公開日、公開URL、メモを管理
- 記事、Substack投稿はタイトル、ステータス、公開日、公開URL、メモを管理
- フォロー/フォロワーは発信者名、URL、ジャンル、星1から3の評価、フォロー日、観察メモを管理
- アイデア保管ではThreads候補、note候補、X候補、Substack候補などを分類
- 全ページで新規追加、編集、削除、検索
- ブラウザのローカルストレージに自動保存
- スマホでも使いやすいレスポンシブ表示

## 開発

```bash
npm.cmd install
npm.cmd run dev -- --host 0.0.0.0 --port 5188
```
